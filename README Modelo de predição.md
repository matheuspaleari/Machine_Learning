# 🧠 Repositório de Treinamento em Machine Learning

Bem-vindo ao meu repositório de estudos e práticas em Machine Learning! 🚀 

Este espaço é dedicado a armazenar meus projetos, experimentos e exercícios de classificação, recomendação e análise de dados utilizando Python e bibliotecas populares de aprendizado de máquina.

## 📁 Estrutura do Repositório e Projetos

Aqui você encontrará diversos Jupyter Notebooks (`.ipynb`) e bases de dados (`.csv`) separados por temas:

* **🚗 Outros Modelos Preditivos**
    * `modelo de predição.ipynb`: Estudo e modelagem baseada em dados de plantas.
      
* **⚙️ Fundamentos e Bibliotecas**
    * `SKLearning.ipynb`: Exercícios práticos focados na biblioteca Scikit-Learn para algoritmos de classificação.

## 🛠️ Tecnologias Utilizadas

Durante o desenvolvimento destes projetos, utilizei as seguintes ferramentas e bibliotecas:

* **Linguagem:** Python
* **Manipulação de Dados:** Pandas, NumPy
* **Machine Learning:** Scikit-Learn (classificação, regressão, redução de dimensionalidade)
* **Ambiente:** Jupyter Notebook
